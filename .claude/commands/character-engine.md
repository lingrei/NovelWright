## 0. 主角设计

### 0.1 代入模式 vs 观赏模式

- **代入模式（成为他）：** 主角半透明，读者通过他的眼睛看世界
- **观赏模式（看着他）：** 主角有独立人格，读者欣赏/恐惧/嫉妒他

### 0.2 方法论推导

- 核心能力/资源是什么？
- 方法如何与故事冲突**精确匹配**？
- 主角的弱点是什么？弱点创造意外脆弱时刻。

## 1. 角色存在的理由

**"读者为什么在乎她/他？"**

### 1.1 从核心渴望推导

- 如果渴望需要"打破"什么 → 角色必须有值得被打破的东西
- 如果渴望需要"跨越"什么 → 角色身上必须有那条线
- 如果渴望需要"观看变化" → 角色需要明确的起点状态

### 1.2 推导外表、性格、处境

**外表** — 不是"长什么样"，是"什么特征让读者记住这个角色"

```
- 面容: [特征] — 传达: [信号]
- 身材: [特征] — 传达: [信号]
- 气质: [特征] — 传达: [信号]
- 辨识度: [X/10]
```

**性格** — 不是"什么性格"，是"什么行为让故事更有张力"

```
- 表面: [表现] — 制造的障碍: [...]
- 内在: [里面] — 暴露条件: [...]
- Gap: [表面 vs 内在的落差]
```

**处境** — 不是"什么身份"，是"身份如何放大故事张力"

```
- 身份: [社会角色]
- 张力来源: [身份如何制造冲突]
- 删除测试: [没有这个身份损失什么？]
```

**物理存在感档案**

```
| 特征 | 物理属性 | 叙事功能 | 变化轨迹 |
|------|---------|---------|---------|
```

## 2. 变化潜力扫描

扫描所有可以变化的原材料，为W3提供选择空间：

- **心理弱点:** 最害怕什么？什么能让角色动摇？
- **心理渴望:** 最想要什么？得不到或不允许自己要的？
- **认知盲区:** 对世界/自己有什么错误认知？
- **关系张力:** 与其他角色之间有什么未解决的张力？
- **身心关系:** 身体和心理通常一致还是分裂？

---

## 3. 独特性检验

- 把角色的戏份给另一个角色，故事损失什么？
- 角色必须有**只有ta才能提供**的东西

## 4. 深度检验

- 他独处时做什么？
- 他最脆弱的时候是什么样子？
- 如果删掉主线剧情，他还有趣吗？
- 关于他的任何问题，你都能回答吗？
