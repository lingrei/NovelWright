# IDEA ENGINE: 灵感启发

> **Phase 0 是纯对话阶段。不产出文件，只产出核心前提。**

---

## 1. 起点探测

问用户：

- 你已经有什么了？（一个画面？一个角色？一种感觉？一个"如果..."？）
- 你想让读者感受到什么？（紧张？心碎？释然？恐惧？温暖？）
- 有没有参考作品？（不一定是同类型——可以是"X的氛围 + Y的结构"）

---

## 2. 核心渴望提取

从用户的回答中提取：

**"读者在深夜搜索时真正渴望的东西是什么？"**

- 这不是类型（"悬疑"）
- 这不是设定（"校园"）
- 这是**情感体验**（"看着一个完美的人慢慢碎裂"）

### 渴望深挖
- 为什么会渴望这个？心理根源？
- 现实中被什么压抑了？
- 满足这个渴望的最佳形态是什么？

---

## 3. 叙事机制草案

从核心渴望推导：**用什么机制来兑现这个承诺？**

- 时间压力？信息差？能力系统？关系博弈？
- 这个机制如何创造持续的张力？
- 它的规则是什么？

---

## 4. 退出条件

当以下都确定时，Phase 0 完成：
- [ ] 核心渴望（一句话）
- [ ] 叙事机制草案
- [ ] 用户说 "好，就这个"

→ 进入 W1（世界构建）
