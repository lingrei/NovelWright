# PROSE AUDITOR: 审计协议

---

## Section A: Batch即时审计（6层Checklist）

| # | 审计层 | 检查什么 |
|---|--------|----------|
| ① | **格式规范** | 零英文混入、正确标点、无概念标签、无过渡预告 |
| ② | **篇幅与对话** | 篇幅充实、对话密度匹配场景、无连续无对话段 |
| ③ | **感官密度与均衡** | ≥3个感官细节/Beat、触觉必填、反视觉主导、环境锚定 |
| ④ | **Camera_OS** | 可拍摄、正面断言、无抽象（意象比喻除外）、Shutter Cut结尾 |
| ⑤ | **边界约束** | 强度边界、禁区、状态流、Dilation密度合规 |
| ⑤.5 | **事实一致性** | 与_state.yaml的4D状态无矛盾 |
| ⑥ | **结构** | 与上一Batch的Continuity |

**失败处理：** 定位问题 → 即时修复 → 重新审计

## Section B: Chunk综合审查（5层）

| Layer | 审查层 | 检查什么 |
|-------|--------|----------|
| **1** | 格式规范 | Shutter Cut、正面断言、无概念标签 |
| **2** | 逻辑一致性 | Beat覆盖、强度边界、禁区、状态流 |
| **3** | 散文打磨 | Dilation节奏、感官多样性 |
| **4** | 角色一致性 | 表面性格、Gap条件、变化阶段 |
| **5** | Voice一致性 | Voice全篇一致、角色语言DNA、无范例复制 |
