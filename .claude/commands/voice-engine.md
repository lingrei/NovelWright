# VOICE ENGINE

## Section 1: 叙事Voice定义（W1阶段使用）

从核心前提推导叙事声音：

### 1.1 叙事者类型
- **角色视角 (Close POV):** 读者通过角色的眼睛看世界
- **隐身作者:** 叙事者不存在于故事中，但有美学立场
- **全知叙事:** 可自由进入任何角色内心

### 1.2 语言风格三维度

| 维度 | 光谱 |
|------|------|
| **节奏** | 短促利落 ←→ 绵长流动 |
| **温度** | 冷静克制 ←→ 热烈直接 |
| **文学性** | 口语化 ←→ 高度文学 |

### 1.3 镜头模式

| 模式 | 描写什么 | 适用场景 |
|------|---------|---------|
| **PHYSICAL** | 可被摄像机记录的外部行为 | 动作场景、物理互动 |
| **SUBJECTIVE** | 角色内心感受、思维 | 心理转折、内心独白 |
| **NARRATOR** | 叙事者评论、时间跳跃 | 过渡、背景交代 |

### Voice产出格式（写入_Setting.md）
```
- 叙事者: [类型]
- 语言风格: [节奏] + [温度] + [文学性]
- 允许的镜头模式: [列表]
- Voice规则（三元组）:
  | 原则 | 验证标准 | 违规标志 |
```

## Section 2: 角色语言DNA（W2阶段使用）

为每个角色设计独特的说话方式：

- **词汇层:** 用什么词？口头禅？专业术语？
- **句式层:** 说话长还是短？完整句还是碎片？
- **情绪表达:** 高兴/愤怒/紧张时说话方式如何变化？
- **语言渐进:** 随角色变化，说话方式如何演变？

### 语言DNA产出格式
```
- 词汇特征: [...]
- 句式特征: [...]
- 情绪变化: [高兴→... / 愤怒→... / 紧张→...]
- 语言渐进: 起点: [...] → 中期: [...] → 终点: [...]
- 称呼变化: [对主要角色的称呼轨迹]
```
