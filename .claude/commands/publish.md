# PUBLISH: 发布策划

---

## 1. 导出

使用 `.agent/scripts/export_clean.py` 生成纯净文本版本。

## 2. 平台适配

| 平台 | 格式要求 | 注意事项 |
|------|---------|---------|
| **Kindle** | .epub / .mobi | 需要章节目录 |
| **Web连载** | 分章markdown | 每章末尾加钩子 |
| **实体出版** | .docx | 标准稿件格式 |

## 3. 元数据准备

- 标题 / 副标题
- 摘要（150字 / 500字两个版本）
- 标签 / 分类
- 封面概念描述（供AI图像生成参考）
